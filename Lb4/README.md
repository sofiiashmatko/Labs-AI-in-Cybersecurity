### Local LLM Models with Ollama

### Мета роботи

Метою роботи є локальне розгортання великих мовних моделей за допомогою Ollama, виконання базових сценаріїв використання мовних моделей та порівняння результатів роботи двох моделей у задачах чат-бота, генерації тексту та multimodal data.

### Використані моделі

* qwen3.5:0.8b
* gemma3:4b

### Завантаження моделей за допомогою команд:

ollama pull qwen3.5:0.8b

ollama pull gemma3:4b


### Реалізовані сценарії

### а) Chat Bot

Було реалізовано чат-бот, який надсилає однаковий текстовий запит до двох моделей.

Код наведено у файлі: [chatbot.py](chatbot.py)

### Запит: What is artificial intelligence?

Результат наведено у вигляді текстового файлу: [result_a_chatbot.txt](result_a_chatbot.txt)


### Аналіз:

### qwen3.5:0.8b

Модель сформувала структуровану коротку відповідь із поясненням:

* визначення AI
* принцип роботи
* приклади використання

Відповідь компактна та технічна.

### gemma3:4b

Модель надала розширену відповідь:

* weak AI / strong AI
* machine learning
* deep learning
* приклади галузей використання

Відповідь більш детальна та академічна.

### Висновок: gemma3:4b забезпечує більш повне пояснення, qwen3.5:0.8b працює швидше.

### б) Text Generation

Було реалізовано генерацію тексту.

### Запит: Write a short text about a healthy lifestyle.

Код наведено у файлі: [text-generate.py](text-generate.py)

### qwen3.5:0.8b

Модель створила текст у формі структурованого списку:

### Висновок: qwen3.5:0.8b краще підходить для формальних відповідей, gemma3:4b — для природної генерації тексту.

### Аналіз:

### qwen3.5:0.8b

Модель детально описала:

* дві жирафи
* розташування тварин

* освітлення

### gemma3:4b

Модель сформувала більш природний опис:

* setting

### Висновок: обидві моделі правильно розпізнали зображення. qwen3.5:0.8b дала більш деталізований опис, gemma3:4b - більш природний стиль.

### Загальний висновок

Порівняння результатів показало, що Qwen3.5 qwen3.5:0.8b працює швидше та потребує менше ресурсів, а Gemma gemma3:4b забезпечує більш точні, розгорнуті та стилістично якісні відповіді. Обидві моделі придатні для локального використання, але вибір залежить від вимог до швидкості та якості результату.
